# screw

Very simple command line web crawler with Go.

## Requirements

- [Go](https://golang.org/)

## Installation

### Homebrew

```bash
$ brew install muniere/triv/screw
```

### Manual

```bash
# clone
$ git clone git@github.com:muniere/screw.git

# install
$ cd screw
$ make && make install
```
